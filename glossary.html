Glossary

(Use ctrl-f/cmd-f to search for the word you need help with)

  1. 0 matrix = A matrix of 0â€™s, (all elements = 0).
  2. 2x2 = A square matrix with 2 rows and 2 columns, often used for modelling system in x and y coordinates only.
  3. 3x3 = A square matrix with 3 rows and 3 columns, often used for modelling a 3 dimensional system, x, y and z.
  4. Associative = This is a property of matrices that means (AxB)xC=Ax(BxC).
  5. Column = A set of numbers in a given vertical of a matrix make up a column.
  6. Determinant = Often denoted det(A) or |A|, this is the volume scaling factor for the transformation represented.
  7. Eigenvector = These are vectors that remain unchanged by a given transformation.
  8. Eigenvalue = This is the associated scaling factor for the eigenvector.
  9. Element = An element is the individual entry in the matrix, any one value.
  10. Enlargement = With its dimensions unchanged by a transformation, a shapes area is increased by a scale factor.
  11. Identity = This is any sqaure matrix (most commonly 2x2 and 3x3) with its leading diagonals elements having value of 1, all other elements being 0.
  12. Invariant = When a point or set of points undergoes a transformation, an invariant point is one that does not change its position. A line of invariant points is formed by a lien of points which do not move as a result of being on the line of reflection. An invariant line is one that connects the transformed point on a shape to its original coordinate.
  13. Matrix = A matrix is rectangular array of values or collection of column vectors that allows for simple data manipulation and modelling of geometric operations.
  14. Plane(s) = A plane is a 2 dimensional surface, often applied to describing the combination of two of the coordinated axis
  15. Row = A set of numbers in a given horizontal of a matrix make up a row.
  16. Shear = If a non-zero value replaces a zero valued element in the identity matrix, then the shape will stretch uniaxial.
  17. Stretch = A shapes dimension will be altered with a stretch, one direction of increase occurring, movement relative to a point or line of origin.
  18. Subscript = The positon of an element in a matrix can be described with subscript notation. The shape of a matrix is described by m x n, it having m rows and n columns, so an element can be described aij, I and j being the respective row and column for a given element.
  19. Transformation = This refers to the movement and translation of objects or a set of coordinates in the coordinate plane.
  20. Transpose = The transpose of a matrix creates a new one for which the rows are the columns of the original. Denoted by AT.
  21. Vector = A series of numbers in a row or column that describe the displacement of a point, a direction or acceleration. (Commonly values ordered relative to x then y then z (if applicable)).
